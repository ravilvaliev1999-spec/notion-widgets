# notion-widgets

Виджеты для Notion. Актуальная версия виджета «Файлы задачи» — **task-files-widget-v18.html**.

## Что нового в v18

**1. Мгновенная загрузка виджета в Notion.**
Раньше при старте виджет вставлял JSONP-скрипты к `script.google.com` (проверка связи и автопривязка). Такие скрипты задерживают событие `load` iframe, а Notion крутит спиннер, пока `load` не сработает — «холодный» Apps Script отвечает 2–10 секунд. Теперь до события `load` виджет не делает ни одного сетевого запроса: интерфейс и список файлов рисуются сразу из localStorage, автопривязка и проверки стартуют после.

**2. Скачивание файлов работает и происходит мгновенно.**
Раньше скачивание шло через скрытый iframe на `drive.google.com/uc?export=download` — в песочнице Notion браузер молча блокирует такие загрузки. Теперь клик по файлу скачивает его через Blob + `<a download>`: без вкладок, без всплывающих окон, ровно в том формате, в котором файл был загружен.

**3. Файл с компьютера добавляется мгновенно.**
Карточка появляется сразу, загрузка в Drive идёт в фоне (подпись «загружается в Drive…»). Всё это время файл уже можно скачивать — байты держатся локально. Файлы до 2,5 МБ сохраняются в localStorage и скачиваются мгновенно даже после перезагрузки страницы.

## Установка / обновление

1. **Виджет.** Включите GitHub Pages для этого репозитория (Settings → Pages → Deploy from branch → main). В Notion вставьте embed-ссылку:
   `https://<ваш-логин>.github.io/notion-widgets/task-files-widget-v18.html`
   Для уже привязанных блоков сохраните хвост `#task=…` из старой ссылки.

2. **Apps Script (обязательно для скачивания из Drive).** Откройте свой проект на [script.google.com](https://script.google.com), замените код содержимым файла [`apps-script/Code.gs`](apps-script/Code.gs) и **переразверните**: Deploy → Manage deployments → ✏ → Version: **New version** → Deploy. Без новой версии рабочий URL продолжит отдавать старый код, и у виджета не будет команды `download`.

3. **Автопривязка (по желанию).** В Apps Script: Project Settings → Script properties → добавьте `NOTION_TOKEN` с токеном интеграции Notion, а сами страницы подключите к интеграции через Connections.

`SECRET` в Code.gs должен совпадать с `WIDGET_KEY` в HTML виджета (уже совпадает).

## Ограничения

- Загрузка файла с компьютера через Apps Script — до 15 МБ; скачивание через скрипт — до 30 МБ (больше — открывайте в Drive по кнопке ↗).
- Список файлов хранится в localStorage браузера: он не синхронизируется между устройствами (сами файлы лежат в Drive в папке «Notion — файлы задач/<задача>»).
